# GoNext

Минимальное приложение на Expo Router + TypeScript с использованием React Native Paper.

## Технологии

- Expo SDK 54
- Expo Router 4.0
- React Native 0.81
- React 19.1
- React Native Paper 5.12
- TypeScript 5.3

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Затем выберите платформу:
- `a` для Android
- `i` для iOS
- `w` для Web

## Структура проекта

- `app/` - директория с экранами приложения (Expo Router)
  - `_layout.tsx` - корневой layout с React Native Paper Provider
  - `index.tsx` - экран Home с AppBar, текстом, кнопкой и Snackbar
